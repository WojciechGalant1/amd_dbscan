# AMD-DBSCAN – Adaptive Multi-Density DBSCAN

Implementacja algorytmu AMD-DBSCAN w języku Python, przygotowana na potrzeby projektu z analizy danych.  
Repozytorium zawiera:

- implementację algorytmu AMD-DBSCAN,
- klasyczny DBSCAN jako punkt odniesienia,
- moduł do eksperymentów i wizualizacji,
- przykłady użycia.

---

## Wymagania

- Python **3.10+**
- pip lub conda

### Wymagane biblioteki:

- `numpy`
- `scikit-learn`
- `matplotlib`
- `scipy`
- `pandas`
